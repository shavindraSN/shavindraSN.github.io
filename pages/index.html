Overview

Examen API Reference

Welcome to Examen's API Documentation! Examen is powerful, user-friendly online Multiple Choice Questions (M.C.Q) based quiz system focus to help G.C.E Advanced level students in Sri Lanka.

Examen is built in a way that anyone can extend and improve the system or build completely new set of functionalities. Examen can be used to build another set of applications that either improve existing functionalities or completely new functionalities. Our API Documentation describes existing functionalities and services provided for you. You can create your own application on top of Examen API. There are thousands of new possibilities that you may imagined. This API provide basic functionalities and services that you can use as building blocks of your own application. We outlined some basic functionalities of our API below

Before diving into Examen's API resources, you might want to refer to following features of the Examen API

  • Retrieve questions in the question bank
  • Add new questions to question bank
  • Get random questions set for given time and subject
  • Get all questions
  • Get all subjects
  • Managing users
  • Creating Papers using Excel sheet
  • Upload question individually
